### Завдання 1 та 2
Завантажити з бібліотеки MATLAB і відобразити на екрані ПК кілька зображень у різних форматах, а також завантажити власне зображення з довільного каталогу.

Було використано стандартні зображення середовища та власне завантажене фото.

```
% 1. Завантаження та відображення зображень з бібліотеки MATLAB
img_peppers = imread('peppers.png');
img_cameraman = imread('cameraman.tif');

figure('Name', 'Бібліотечні зображення');
subplot(1, 2, 1); imshow(img_peppers); title('peppers.png');
subplot(1, 2, 2); imshow(img_cameraman); title('cameraman.tif');

% 2. Завантаження власного зображення
filename = 'foto.jpg';
my_img = imread(filename);

figure('Name', 'Оригінальне зображення');
imshow(my_img); title('Власне зображення');

```

### Завдання 5
З використанням процедури imhist(I) знайти гістограми розподілу яскравостей завантажених зображень і відобразити їх на екрані ПК.

Для коректної побудови гістограми кольорового зображення (RGB) його попередньо перетворено у відтінки сірого за допомогою rgb2gray.

```
% 5. Побудова гістограми розподілу яскравостей
% Функція imhist працює з 2D-матрицями. Для кольорового (RGB) зображення 
% спочатку конвертуємо його у відтінки сірого.
img_gray = rgb2gray(my_img);
figure('Name', 'Гістограма');
imhist(img_gray);
title('Гістограма розподілу яскравостей (градації сірого)');

```

### Завдання 6 та 7
З використанням процедури imadjust(I) виконати контрастування зображень та відобразити результат.

Функція автоматично обрізає по 1% найсвітліших і найтемніших пікселів, розтягуючи гістограму.
```
 6 та 7. Контрастування та відображення результату
% Функція imadjust за замовчуванням розтягує гістограму, обрізаючи по 1%
% найсвітліших і найтемніших пікселів, що підвищує контраст.
img_adj = imadjust(img_gray);

figure('Name', 'Контрастування');
subplot(1, 2, 1); imshow(img_gray); title('Оригінал (сірий)');
subplot(1, 2, 2); imshow(img_adj); title('Підвищена контрастність');
```

## Висновки
Під час виконання самостійної роботи закріплено базові навички обробки зображень у MATLAB. Успішно реалізовано алгоритми читання та запису графічних файлів, вилучення метаданих. Досліджено вплив градаційних перетворень на візуальне сприйняття зображення: встановлено, що використання функції imadjust дозволяє розширити динамічний діапазон яскравостей, а imcomplement — коректно формує негатив шляхом інверсії значень пікселів.

## Контрольні запитання 
1. Що таке «гістограма розподілу яскравостей»?

    Це статистичний графік, який показує кількість пікселів кожного рівня яскравості на зображенні (від 0 до 255 для стандартного 8-бітного формату).

2. Що таке «контрастність зображення»?

    Це міра різниці між найсвітлішими та найтемнішими ділянками зображення. Високий контраст означає наявність глибоких тіней та яскравих світлих ділянок із чіткими межами.

3. Як при контрастуванні змінюється гістограма розподілу яскравостей зображення?

    При підвищенні контрасту гістограма розтягується на весь доступний діапазон яскравостей. Піки значень стають ширшими та віддаляються один від одного.

4. Як за необхідності зменшити контрастність зображення?
   
    Потрібно звузити діапазон яскравостей. У MATLAB це робиться за допомогою функції imadjust, де діапазон вихідних значень задається вужчим за вхідний, наприклад: imadjust(I, [0 1], [0.2 0.8]).

5. Як одержати негативне зображення?
   
    Необхідно інвертувати яскравість кожного пікселя. У середовищі MATLAB для цього використовується вбудована функція imcomplement().

## Мета
Метою даної лабораторної роботи є дослідження ефективності
існуючих методів фільтрації зображень, що були спотворені шумами різних

### Завдання 1-2
Завантажити з бібліотеки MATLAB кілька тестових зображень. Відобразити вихідні зображення на екрані ПК.

У цьому завданні було використано функцію imread для завантаження стандартних зображень та im2double для приведення їх до діапазону $[0, 1]$, що необхідно для коректної математичної обробки.
```
%% 1. Завантаження тестових зображень
I1 = im2double(imread('cameraman.tif'));
I2 = im2double(imread('coins.png'));

%% 2. Відображення вихідних зображень
figure('Name', 'Вихідні зображення');
subplot(1,2,1); imshow(I1); title('Cameraman');
subplot(1,2,2); imshow(I2); title('Coins');
```

### Завдання 3-4
Здійснити процедуру зашумлення зображення нормальним білим шумом і імпульсною перешкодою з різною щільністю. Відобразити зашумлені зображення.

Для моделювання спотворень використано функцію imnoise. Гаусів шум додає випадкові значення до кожного пікселя, тоді як імпульсний ("salt & pepper") замінює випадкові пікселі на екстремальні значення (чорне/біле).

```
%% 3-4. Зашумлення та відображення
% Гаусів шум (нормальний білий)
I1_gauss = imnoise(I1, 'gaussian', 0, 0.02); 
% Імпульсна перешкода (salt & pepper)
I1_sp = imnoise(I1, 'salt & pepper', 0.05);

figure('Name', 'Зашумлені зображення');
subplot(1,2,1); imshow(I1_gauss); title('Гаусів шум (0.02)');
subplot(1,2,2); imshow(I1_sp); title('Імпульсний шум (0.05)');
```

### Завдання 5-6
Виконати фільтрацію вихідних зображень лінійними фільтрами з використанням віконних фільтрів низьких і високих частот. Відобразити зображення.

Низькочастотна фільтрація (ФНЧ) виконується для згладжування, а високочастотна (ФВЧ) — для підкреслення контурів.

```
%% 5-6. Лінійна фільтрація 
% Низькі частоти (розмиття)
h_low = fspecial('average', [3 3]);
I_low = imfilter(I1, h_low);

% Високі частоти (підкреслення контурів/різкість)
h_high = [0 -1 0; -1 5 -1; 0 -1 0]; 
I_high = imfilter(I1, h_high);

figure('Name', 'Лінійна фільтрація вихідного зображення');
subplot(1,2,1); imshow(I_low); title('Низькочастотна (Average)');
subplot(1,2,2); imshow(I_high); title('Високочастотна (Unsharp)');
```

## Висновки
У ході виконання роботи було досліджено методи просторової фільтрації. Встановлено, що лінійні осереднювальні фільтри ефективно зменшують Гаусів шум, але спричиняють розмиття контурів. Медіанний фільтр продемонстрував найкращу ефективність у боротьбі з імпульсним шумом, повністю видаляючи «білі та чорні точки» без значної втрати чіткості. Адаптивний фільтр Вінера показав високу якість придушення Гауссового шуму за рахунок врахування локальних характеристик зображення.

## Контрольні запитання
Поясніть процедуру лінійної фільтрації зображення віконним фільтром.
Це процес обчислення нового значення пікселя як зваженої суми значень пікселів у межах вікна (маски). Маска ковзає по всьому зображенню, виконуючи математичну операцію згортки.

Чому відрізняються “вікна фільтрів” для низькочастотної і високочастотної фільтрації?
Вікна ФНЧ мають додатні коефіцієнти, що усереднюють яскравість (інтегрування). Вікна ФВЧ мають від'ємні та додатні значення, що реагують на різку різницю між сусідніми пікселями (диференціювання).

Як змінюється вигляд зображення після його фільтрації низькочастотними фільтрами?
Зображення стає розмитим, дрібні деталі та шум зникають, контури об'єктів стають менш чіткими.

Як змінюється вигляд зображення після його фільтрації високочастотними фільтрами?
Підкреслюються межі об'єктів та дрібні деталі, зображення виглядає більш різким, але при цьому може посилитися візуальний шум.

Для перешкод якого характеру більше підходить медіанна фільтрація, і чому?
Найкраще підходить для імпульсного шуму («сіль та перець»). Оскільки імпульсний шум створює екстремальні значення (викиди), медіана серед значень у вікні дозволяє повністю ігнорувати ці викиди та обрати реальне значення сусіднього пікселя.

# Лабораторна робота № 3
## Мета роботи
Метою даної лабораторної роботи є дослідження методів
відновлення перекручених зображень як за відсутності шумів, так і з ними.

### Завдання 1
Завантажити тестове зображення та підготувати його до обробки.
Було створено еталонне зображення (шахівниця) та перетворено його у формат double. Це необхідно для уникнення помилок округлення при складних математичних операціях відновлення.

```
%% 1. Завантаження та підготовка зображення
I = checkerboard(8); % Створення тестового зображення (шахівниця)
I = im2double(I);
figure; imshow(I); title('Оригінальне зображення');
```

### Завдання 2
Змоделювати лінійні спотворення (розмиття) та додати шум.
Створено функцію розмиття (PSF), яка імітує ефект лінійного руху (тремтіння камери). Після застосування розмиття за допомогою imfilter, до зображення було додано білий гаусів шум за допомогою imnoise.

```
%% 2. Моделювання спотворення (Розмиття + Шум)
% Створюємо функцію розмиття (PSF) - лінійний рух
PSF = fspecial('motion', 21, 11);
% Застосовуємо розмиття
blurred = imfilter(I, PSF, 'conv', 'circular');

% Додаємо білий гаусів шум
noise_mean = 0;
noise_var = 0.0001;
blurred_noisy = imnoise(blurred, 'gaussian', noise_mean, noise_var);

figure; imshow(blurred_noisy); title('Спотворене зображення (Розмиття + Шум)');

```

### Завдання 3
Виконати відновлення зображення методом інверсної фільтрації.
Використано функцію deconvwnr зі значенням шуму рівним 0. Оскільки інверсна фільтрація не враховує наявність завад, шум підсилився, що призвело до появи сильних артефактів і спотворення результату.

```
%% 3. Відновлення методом інверсної фільтрації 
luc1 = deconvwnr(blurred_noisy, PSF, 0);
figure; imshow(luc1); title('Інверсна фільтрація (без урахування шуму)');
```

### Завдання 4
Виконати відновлення зображення за допомогою фільтра Вінера.
Для коректної роботи фільтра було розраховано відношення сигнал/шум (NSR). Передача цього параметра у функцію deconvwnr дозволила алгоритму збалансувати відновлення різкості та пригнічення шуму.

```
%% 4. Відновлення за допомогою фільтра Вінера
% Розрахунок відношення сигнал/шум (NSR)
snr = noise_var / var(I(:));
luc2 = deconvwnr(blurred_noisy, PSF, snr);
figure; imshow(luc2); title('Фільтр Вінера (NSR)');
```

### Завдання 6
Виконати відновлення зображення ітераційним методом Люсі-Річардсона.
Застосовано функцію deconvlucy. Кількість ітерацій було обмежено до 20, щоб запобігти надмірному підсиленню шумової складової на пізніх етапах алгоритму.

```
%% 6. Відновлення методом Люсі-Річардсона 
% Ітераційний метод
luc4 = deconvlucy(blurred_noisy, PSF, 20); % 20 ітерацій
```

### Завдання 7
Виконати зашумлення початкового зображення та порівняти результати відновлення при сильних спотвореннях.
Початкове зображення було зашумлено з вищою дисперсією, після чого до нього застосовано сильніше розмиття. Експеримент довів, що при правильній оцінці відношення сигнал/шум фільтр Вінера здатен ефективно відновлювати деталі навіть за умов сильних спотворень.

```
%% 7. Додаткове завдання: Експерименти з шумом
% 7.1 Створення зображення з сильним шумом (без розмиття для тесту)
I_noisy_only = imnoise(I, 'gaussian', 0, 0.01); 

% 7.2 Повторення пунктів 2-6 для нового стану шум)
PSF_strong = fspecial('motion', 40, 45);
blurred_strong = imfilter(I, PSF_strong, 'conv', 'circular');
blurred_noisy_strong = imnoise(blurred_strong, 'gaussian', 0, 0.005);

% Відновлення Вінером для порівняння 
snr_strong = 0.005 / var(I(:));
res_strong = deconvwnr(blurred_noisy_strong, PSF_strong, snr_strong);

figure;
subplot(1,2,1); imshow(blurred_noisy_strong); title('Сильне спотворення (п.7)');
subplot(1,2,2); imshow(res_strong); title('Відновлення при сильному шумі');
```

## Висновки
У ході лабораторної роботи було досліджено математичні модели формування та спотворення зображень (розмиття та додавання шуму). Отримано практичні навички роботи з алгоритмами деконволюції в MATLAB.
Встановлено, що класична інверсна фільтрація є непридатною для зашумлених зображень через підсилення завад. Найкращі результати візуальної якості продемонстрували фільтр Вінера (за умови точної оцінки рівня шуму) та метод Люсі-Річардсона, які здатні ефективно відновлювати втрачену різкість із мінімізацією артефактів.

## Контрольні запитання
1. Поясніть процес формування зображення.

Процес формування цифрового зображення полягає у перетворенні неперервного оптичного сигналу у дискретну матрицю чисел. Він включає проекцію світла через лінзу на сенсор, дискретизацію (розбиття на пікселі) та квантування (перетворення інтенсивності світла на цифрове значення).

2. Опишіть модель формування зображень та лінійних перекручень.

Математична модель описується рівнянням $g(x,y) = h(x,y) * f(x,y) + \eta(x,y)$, де $f(x,y)$ — ідеальне зображення, $h(x,y)$ — функція розсіювання точки (PSF), що відповідає за розмиття, $*$ — операція згортки, а $\eta(x,y)$ — адитивний шум.

3. Поясніть процес відновлення перекрученого зображення.

Відновлення — це розв'язання оберненої задачі для отримання оцінки, максимально близької до оригіналу. Процес включає оцінку функції розсіювання точки (PSF), виконання деконволюції (оберненої згортки) за допомогою спеціальних фільтрів та придушення шуму для уникнення його підсилення під час відновлення високих частот.

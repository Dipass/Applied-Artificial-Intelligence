## Мета роботи
Метою даної лабораторної роботи є набуття знань про існуючи методи стиснення зображень та ознайомитися з основними з них.

### Завдання 1-2.
Було завантажено тестові зображення та здійснено перетворення кольорових форматів у напівтонові. Це необхідно для подальшої математичної обробки їх яскравісної складової.

```
% Завдання 1: Завантаження зображень
imgRGB = imread('peppers.png'); 
imgGray = imread('cameraman.tif'); 

figure;
subplot(1,2,1); imshow(imgRGB); title('Завдання 1: Вихідне кольорове');
subplot(1,2,2); imshow(imgGray); title('Завдання 1: Вихідне чорно-біле');

% Завдання 2: Перетворення кольорового в чорно-біле
imgConverted = rgb2gray(imgRGB);
figure; 
imshow(imgConverted); 
title('Завдання 2: Перетворене в чорно-біле');

```

### Завдання 3-4. 
За допомогою функції dct2 отримано частотний спектр. Візуалізація спектра в логарифмічному масштабі показує, що основна енергія зображення зосереджена у лівому верхньому куті матриці. Наступне застосування функції idct2 дозволило без втрат відновити вихідне зображення.
```
% Завдання 3: Дискретне косинусне перетворення (ДКП)
I = im2double(imgConverted); 
J = dct2(I);
figure; 
imshow(log(abs(J)), []); 
colormap(gca, jet(64)); colorbar;
title('Завдання 3: ДКП-спектр (логарифмічний масштаб)');

% Завдання 4: Відновлення за ДКП-спектром
I_reconstructed = idct2(J);
figure; 
imshow(I_reconstructed); 
title('Завдання 4: Відновлене зображення (без втрат)');
```

### Завдання 5-6. 
Проведено квантування результатів ДКП. Ця процедура ділить кожен коефіцієнт на крок N, округлює до найближчого цілого і знову множить на N. У результаті дрібні значення перетворюються на нулі, а більші значення групуються за дискретними рівнями. Спектр стає «розрідженим».
```
% Завдання 5: Квантування результатів ДКП
N = 0.05; 
J_quantized = N * round(J / N);

% Завдання 6: Відображення квантованого спектра
figure; 
imshow(log(abs(J_quantized)), []); 
colormap(gca, jet(64)); colorbar;
title(['Завдання 6: Квантований ДКП-спектр (N = ', num2str(N), ')']);
```

### Завдання 7
При відновленні зображення з квантованого спектра за допомогою idct2 спостерігається втрата якості. При великих значеннях кроку квантування N зображення втрачає чіткість, розмивається, а навколо різких контурів з'являються артефакти у вигляді хвиль.

```
% Завдання 7: Відновлення зображення за квантованим спектром
Iq_reconstructed = idct2(J_quantized);
figure; 
imshow(Iq_reconstructed); 
title(['Завдання 7: Відновлене з квантованого спектру (N = ', num2str(N), ')']);
```

### Завдання 9
Квантування самих пікселів зображення замість ДКП-коефіцієнтів не дає аналогічного результату. Результат — ефект постеризації (поява різких меж між кольорами замість плавних переходів). Стиснення при цьому неефективне, бо просторова надмірність не усувається.

```
% Завдання 9: Квантування вихідного зображення (просторовий домен)
n_space = 0.2; 
I_quantized_space = n_space * round(I / n_space);
figure; 
imshow(I_quantized_space); 
title('Завдання 9: Квантоване вихідне зображення');
```

## Висновок
### Яка мета досягається квантуванням коефіцієнтів ДКП?  (Завдання 8)
Головна мета — стиснення даних (наприклад, у форматі JPEG). Велику кількість нулів можна дуже ефективно стиснути алгоритмами архівації, значно зменшивши розмір файлу при збереженні прийнятної візуальної якості.

### Які недоліки ви бачите в стисненні зображень із використанням його ДКП і квантування коефіцієнтів ДКП? (Завдання 10)

Блокові артефакти: при сильному квантуванні на зображенні стають помітні квадрати.

Втрата дрібних деталей: текстури та дрібні елементи розмиваються.

Ефект Гіббса: поява помилкових контурів та хвиль навколо різких переходів.
